📌 프로젝트 개요
사용자는 영양제 성분 및 논문 관리 웹사이트를 개발 중이다.

프론트엔드: HTML + Vanilla JavaScript

백엔드: Java + Spring Boot + JPA + MySQL

Thymeleaf는 사용하지 않으며, REST API 기반으로 UI를 구성

📌 주요 기능 및 시스템 구성

논문-성분-효과 메타분석 시스템

각 논문은 Journal 엔티티로 저장되며, 연구 설계, 맹검, 표본 수, 연구 기간을 바탕으로 신뢰 점수가 산출됨

Journal은 여러 JournalSupplementEffect, JournalSupplementSideEffect 매핑을 가짐

각 매핑은 cohenD, pearsonR, p-value 등을 저장하며, SupplementEffect, SupplementSideEffect 등과 연결되어 종합점수에 영향을 줌

여러 논문을 기반으로 성분-효과 관계에 대한 간이 메타분석 점수를 자동 갱신

공통 인터페이스 추상화

Effect, SupplementEffect, JournalSupplementEffect, Tag 등 유사한 구조를 공통 인터페이스로 추상화

인증/인가 시스템

JWT + Redis 기반 인증

Spring Security 커스터마이징

사용자 역할/레벨에 따른 기능 제한 구현

기타 기능

논문 즐겨찾기 (Bookmark), 알림(Notification), 조회수 추적

문서 페이지에 대한 수정 내역, 메타데이터 기록

사용자 가입, 로그인, 마이페이지 구현

현재 로컬 환경(Intellij(MAVEN), Mysql Workbench, Docker) 에서 개발 중이며, 향후 Oracle Cloud 기반 배포 예정
